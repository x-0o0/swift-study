## Overview

⌜Swift RGB 저널⌟ 은 실험해 보거나 좀 더 깊게 공부한 내용을 정리하고 있습니다.

| 색상 | 키워드 | 설명 |
| --- | --- | --- |
| R | 🧣 Read | 가볍게 읽기 좋음 |
| G | 🌿 Go | 직접 실행해보는 것을 권장 |
| B | 🫐 Blueprint | 참고하면서 개발하기 좋음 |
